# zevHub

## Electric Vehicle Marketplace

### What it is:

> A place for electric vehicle enthusiasts to buy, sell, and trade their vehicle, parts, and accessories.

### Technologies

**Front-End**

- JS
- CSS
- HTML
- React
- Redux
- Bootstrap
- Axios

**Back-End**

- Express
- Node.JS
- Mongoose
- Multer
- Socket.IO
- Amazon S3

**Models**

- User
  - email
  - password
  - starred products
  - products for sale
- Product
  - name
  - description
  - category
  - owner

### Features

- Listings
- Messages - stretch
- Favorites
- Location API - stretch

### MVP

- List products with pictures
- Browse, organise and search products
- Purchase a product and decrement it's inventory
