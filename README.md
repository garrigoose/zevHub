# zevHub

## Electric Vehicle Marketplace

### What it is:

> A place for electric vehicle enthusiasts to buy, sell, and trade their vehicle, parts, and accessories.

### Technologies

**Front-End**

- JS
- CSS
- HTML
- React with Next.js

**Back-End**

- Python
- Flask
- Peewee
- Socket.IO
- Amazon S3

**Models**

- Buyer
  - email
  - password
  - starred products
- Seller
  - email
  - password
  - products for sale
- Admin
  - email
  - password
- Product
  - name
  - description
  - category
  - owner

### Features

- Listings
- Messages
- Favorites
- Location API
